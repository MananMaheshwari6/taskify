# Taskify Frontend

A modern todo list application with a clean and responsive UI.

## Features

- User authentication (login/signup)
- Create, read, update, and delete todos
- Dark mode toggle with local storage persistence
- Timestamps for tasks
- Responsive design for all device sizes
- Improved color scheme and typography

## File Structure

```
frontend/
├── assets/       # Images and other static assets
├── css/
│   ├── styles.css   # Main stylesheet
│   └── variables.css # CSS variables for theming
├── js/
│   └── script.js    # Application logic
└── index.html    # Main HTML file
```

## Technologies Used

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript
- Google Fonts (Poppins and Montserrat)
- Font Awesome icons

## Color Scheme

The application uses a modern color palette with:
- Primary: #3a86ff (Blue)
- Secondary: #8338ec (Purple)
- Accent: #ff006e (Pink)
- Success: #06d6a0 (Green)
- Warning: #ffbe0b (Yellow)
- Danger: #ef476f (Red)

## Dark Mode

The application supports a dark mode that can be toggled using the switch in the header. The theme preference is saved to localStorage.
